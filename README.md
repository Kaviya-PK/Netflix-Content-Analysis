# Netflix Content Analysis 📊

## Project Overview
This project analyzes Netflix content to understand trends in genres, ratings, release years, and country-wise distribution.

## Tools & Technologies
- SQL
- Power BI
- Excel

## Dataset
- Source: Kaggle
- Records: 8,000+
- Data includes movies, TV shows, genres, release year, ratings, and countries.

## Key Analysis Performed
- Content distribution by type (Movies vs TV Shows)
- Top genres on Netflix
- Rating-wise content analysis
- Year-wise content growth
- Country-wise content distribution

## Key Insights
- Movies contribute more than TV shows
- Drama and Comedy dominate Netflix content
- Content production increased significantly after 2015

This project demonstrates practical data analysis skills using SQL queries and Power BI dashboards for business insights.
